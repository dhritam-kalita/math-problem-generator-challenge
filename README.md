# Math Challenge Generator

A simple Python program that generates 10 random math problems (addition, subtraction, multiplication) with operands between 3 and 12. It times how long it takes for the user to solve all problems correctly.

## Requirements

- Python 3.x

## How to Run

1. Run `math-challenge-generator.py`
2. Press ENTER to start
3. Solve each problem by typing the correct answer
4. View your completion time at the end
